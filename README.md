# custom-typing

this ahk file have multiple functions and we are still trying to add more functions! Download the exe file if you do not know how to execute it (the exe file might not be updated on time)

# what can it do?
1. replace hotstrings to emojis or special symbols
2. go to websites when you type "\" and the website name
    - there will be a input box when you type \wiki or \google to search faster
3.  wheel of websites (no finished, it might crash)
4.  show clipboard when you press ctrlC, press ctrl + alt to see whole clipboard
5.  help you build web by putting html or css frame when you type "\css or \html setup"
6.  help you import common stuff in python when you type "\py import"
